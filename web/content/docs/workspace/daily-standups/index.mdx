1. **Chronology**: Files must be named in `YYYY-MM-DD.mdx` format.

3. **Daily Requirement**: A standup should be initialized during the first Sync of each day (EST).
4. **Summary**: The "Latest Standup" is a key part of the `wake_up.sh` protocol.
